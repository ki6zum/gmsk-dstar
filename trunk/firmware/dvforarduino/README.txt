DVforArduino 
This is the main controller software running on the Arduino to run the DV for Arduino GMSK board.
It will run on either the Amega328 or the Mega versions of the Arduino boards.


version 0.1
This is the initial release of the source code.  It should build and run on most any flavor of
Arduino board from the ATmega328 based boards all the way to Mega boards.  The code was built
using Arduino 1.0.1
